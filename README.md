🕷️ Spider Cursor Animation

An interactive and visually engaging Spider Cursor Animation built using HTML, CSS, and JavaScript.
The animation follows the mouse movement and creates a dynamic spider-like effect on the screen.

🎯 How It Works

JavaScript tracks mouse movement using event listeners.

CSS styles create the spider design and animation effects.

DOM manipulation updates the spider’s position in real time.
